# Product

## Register

product

## Users

Web developers, QA engineers, and technical operators who need to add or change request headers while debugging a site. They work inside Chrome, often switch between several environments, and expect the extension to stay out of the way until a rule needs attention.

## Product Purpose

RequestKit makes request-header changes fast, understandable, and reversible, whether users need isolated per-site rules or one shared profile across every site. Success means a user can identify the active scope, see which rules will run, make a small edit, and return to the page without leaving their debugging flow.

## Brand Personality

Precise, compact, dependable. The interface should feel like a mature browser utility: calm enough for repeated daily use, clear about scope and state, and responsive without drawing attention to itself.

## Anti-references

Do not resemble a generic SaaS analytics dashboard, a spacious marketing site, or a mobile app enlarged for desktop. Avoid oversized typography and controls, decorative cards, excessive gradients, thick shadows, redundant helper copy, and large empty regions that do not improve scanning.

## Design Principles

1. Keep the user in the debugging flow; primary actions should be reachable without navigation overhead.
2. Make scope and enabled state obvious before decoration or branding.
3. Prefer compact, aligned information density over large containers and generous but unproductive whitespace.
4. Reveal destructive and uncommon actions progressively while keeping common edits direct.
5. Preserve trustworthy behavior across popup, manager window, storage, and background synchronization.
6. Treat “per site” and “all sites” as explicit, mutually exclusive scopes; preserve both configurations when users switch modes.

## Accessibility & Inclusion

Use semantic controls, visible keyboard focus, accessible names, sufficient contrast, reduced-motion support, and layouts that remain usable with zoom or longer hostnames. Color must reinforce state, not carry state by itself.
